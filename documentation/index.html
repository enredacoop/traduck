Buttons

You can change style of buttons in CSS/style.css file and button colors can changed in active color scheme file like lightgreen.css or crimson.css

We have created buttons for hovered and normal states. Looking at the below screenshot you can change button options

Button Usage: circled-button: This class adds rounded corner around a button.
colored-button: This is a standard button class to add background color to button. Background color can be changed in active color scheme file like crimson.css or lightgreen.css
transparent-button: This adds transparent background to the button.
colored-border: This class is used to add white border of 1 pixel around a button.
hovered-colored-button: This class is used to add background color to hovered state of button. Background color can be changed in active color scheme file like crimson.css or lightgreen.css
hovered-colored-border: This class adds colored border in hover state of button. border color can be changed in active color scheme file like lightgreen.css or crimson.css
hovered-transparent-button: This class adds transparent background to hover state of button.

Text Alignments

We tried our best to make everything very simple.So you don't need to refer element in CSS files to change their alignment. Just use one of our defined classes' name. 3 classes are avaiable for alignment center-text ,  left-text  & right-text

Example:

<h1 class="center-text | left-text | right-text" >Your Text Here</h1>

Please use one of the above mentioned classes.
